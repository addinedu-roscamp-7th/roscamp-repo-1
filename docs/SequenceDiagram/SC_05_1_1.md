    activate pickee
    pickee --> shopee: 
    deactivate pickee
    
    shopee -> packee: [ROS2 Topic] Packee 로봇 정보\n(packee/robot_status)
    activate packee
    packee --> shopee: 
    deactivate packee
    
    shopee -> shopee: 전체 로봇\n정보 표시
end
